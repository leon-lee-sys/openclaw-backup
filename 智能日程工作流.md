# 📅 智能日程工作流

> 通过飞书日历管理您的日程

---

## 功能概览

| 功能 | 状态 | 说明 |
|------|------|------|
| 📋 创建日程 | ✅ | 在飞书日历创建会议/事件 |
| 📅 查看日程 | ✅ | 读取飞书日历事件 |
| 📝 创建待办 | ✅ | 创建飞书任务 |
| 🔄 更新日程 | ✅ | 修改日程时间/内容 |
| 🗑️ 删除日程 | ✅ | 删除飞书日历事件 |

---

## 使用方式

### 方式一：直接告诉小燕子

**创建日程：**
```
"帮我创建一个日程：[会议名称]，时间是[日期][时间]，地点[地点]，参加人[姓名]"
```

**查看日程：**
```
"查看明天的日程"
"查看这周的日程"
"查看4月20日的日程"
```

**创建待办：**
```
"创建一个待办：[任务内容]，截止时间是[日期]"
```

---

### 方式二：每日自动提醒

每天晚上8点，我会主动问您第二天有什么安排，您告诉我就行，我会自动帮您创建到飞书日历。

## 📋 飞书日历事件格式

创建日程时，我会自动转换为以下格式：

```json
{
  "summary": "会议名称",
  "description": "会议描述",
  "start_time": "2026-04-17 09:00:00",
  "end_time": "2026-04-17 10:00:00",
  "location": "会议室名称",
  "attendees": ["参与者邮箱或姓名"]
}
```

## ⚙️ 技术实现

使用飞书开放平台API：
- 日历权限：`calendar:calendar:readonly` ✅
- 创建权限：通过lark-calendar skill实现

---

## 💡 使用示例

**三先生说：** "明天上午9点有BX管理内审末次会议，地点学术报告厅"

**小燕子执行：**
```bash
node skills/lark-calendar/scripts/create-event.mjs \
  --title "BX管理内审末次会议" \
  --start "2026-04-17 09:00:00" \
  --end "2026-04-17 10:00:00" \
  --location "学术报告厅"
```

**结果：** ✅ 成功创建到飞书日历

## 📝 使用方法

### 添加新记录
直接告诉小燕子，例如：
- "小燕子，帮我添加一本书《xxx》，作者xxx，类型是xxx"
- "小燕子，帮我收藏一家餐厅，叫xxx，在上海，类型是川菜"
- "小燕子，帮我记录一个想去的旅游目的地：日本东京"

### 查询记录
直接问小燕子，例如：
- "小燕子，我的美食清单有哪些？"
- "小燕子，帮我看看我的书单"
- "小燕子，上海有哪些想去的餐厅？"

### 修改/删除记录
告诉小燕子，例如：
- "小燕子，把第3本书的状态改成已读完"
- "小燕子，删除那个餐厅记录"

## 💡 设计理念

- **录入时间**：每条记录自动标注录入时间，便于追溯
- **评分系统**：统一的评分标准，方便比较和筛选
- **状态跟踪**：记录从"想"到"做"的全过程
- **详细备注**：保留更多个性化信息

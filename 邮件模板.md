# 📧 智能回复模板库

> 通过飞书智能回复QQ邮件的完整流程

---

## 📋 智能回复工作流

| 步骤 | 操作 |
|------|------|
| 1️⃣ | "查看最近邮件" |
| 2️⃣ | "查看第X封" |
| 3️⃣ | "帮我起草回复" + 选择类型 |
| 4️⃣ | 选择/修改回复内容 |
| 5️⃣ | "发送回复" |

---

## ✍️ 智能回复类型（9种）

### 1. ⭐ 专业正式版
> 适用：商务邮件、正式沟通、客户往来

```
尊敬的[对方姓名]：

您好！感谢您的来函。

[根据内容填写]

如有进一步问题，欢迎随时联系。

此致
敬礼

李
```

---

### 2. 📝 简洁明了版
> 适用：日常沟通、简单确认、内部邮件

```
您好！

感谢来信。您的邮件已收到，回复如下：

[具体回复内容]

李
```

---

### 3. 🤗 友好亲切版
> 适用：熟悉同事、客户、轻松场景

```
您好！

谢谢您的来信！看到您的邮件很高兴～

关于您提到的事情：

[具体回复内容]

期待您的回复！

祝好
李
```

---

### 4. ✅ 确认收到
> 适用：收到文件、方案、报价等需要确认的场景

```
您好！

您的[邮件主题/附件]已收到，非常感谢！

我先确认一下内容，有问题会尽快联系您。

如需进一步沟通，请随时联系我。

李
```

---

### 5. 📅 会议确认
> 适用：确认参加/回复会议邀请

```
您好！

确认参加您组织的[会议名称]。

时间：[具体时间]
内容：[如有补充]

如需我准备任何资料，请提前告知，我会做好准备。

李
```

---

### 6. ⏰ 需要延期
> 适用：无法按时完成、需要延长截止日期

```
您好！

关于[任务/项目名称]的截止日期，我需要申请延期。

原因：[简要说明原因，如：需协调更多资源/方案需进一步优化/突发情况等]

预计完成时间：[新日期]

如有问题请随时联系我。

李
```

---

### 7. 🔄 转交他人
> 适用：此事非我负责，需转给相关同事

```
您好！

您的邮件已收悉。

此事由我的同事[姓名]负责，我已经将您的邮件转给他/她。

联系方式：[姓名]：[电话/邮箱]

如有紧急情况也可联系我，我会尽力协助。

李
```

---

### 8. 🙏 委婉拒绝
> 适用：无法满足请求、需礼貌拒绝

```
您好！

非常感谢您的信任与邀请。

关于您提到的[请求内容]，经过慎重考虑，目前我[无法参加/无法满足/条件不允许]。

主要原因是：[简要说明，如：时间冲突/超出当前能力范围/已有其他安排]

希望您能理解，也期待未来有机会合作。

祝顺利！

李
```

---

### 9. 📎 资料请求
> 适用：需要对方提供更多信息/材料

```
您好！

感谢您的来信。

为了更好地[处理/评估/了解]相关事项，请协助提供以下材料：

1. [材料1]
2. [材料2]
3. [其他要求]

如有疑问请随时联系我。

李
```

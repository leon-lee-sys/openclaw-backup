# PPT智能生成工作流

## 支持的PPT类型

| 类型 | 适用场景 | 核心结构 | 图表 |
|------|----------|----------|------|
| 📊 工作汇报 | 周报/月报/季度汇报/年度汇报 | 概述→进展→问题→计划 | 进度对比、完成率环形、趋势折线 |
| 📈 数据分析 | 运营/市场/用户/财务/业务分析 | 背景→指标→趋势→诊断→建议 | 柱状图、折线图、饼图、热力图 |
| 📋 项目立项提案 | 新产品/技术改造/基建/研发立项 | 背景→目标→方案→投资→风险→效益 | 甘特图、预算分解、投资回报曲线 |
| 🎓 培训课件 | 新员工/技能/管理/安全/产品培训 | 目标→大纲→知识点→实操→总结 | 知识结构图、流程图、步骤分解图 |
| 💼 产品介绍 | 产品发布/销售支持/客户演示 | 市场→产品→功能→优势→场景→案例 | 功能对比表、应用场景图、客户案例柱状 |
| 🚀 融资路演 | 天使轮/A轮/B轮/C轮/战略投资 | 团队→市场→方案→模式→运营→规划→融资 | 市场规模柱状、增长曲线、单位经济模型 |
| 📉 经济运行分析 | 年度/季度/行业/区域经济分析 | 宏观→概况→指标→结构→问题→预测→对策 | GDP趋势、产业结构饼图、PMI走势 |
| 🎯 策划方案 | 营销/活动/品牌/危机公关策划 | 背景→受众→策略→创意→执行→预算→评估 | 受众画像、传播路径图、甘特图、预算饼图 |

## 工作流程（4步）

### Step 1：智能识别与大纲生成
- 识别三先生想要的PPT类型
- 根据类型自动匹配专业结构（大纲页数通常7-10页）
- 选择匹配的风格（商务蓝/科技深/学术白/创意紫）

### Step 2：数据图表生成与整合
- 根据PPT类型生成对应的图表数据
- 支持：柱状图、折线图、饼图、环形图、甘特图、散点图、热力图
- 可导入外部Excel数据，自动整合到图表

### Step 3：网络资料深度搜集与整理
- 深度搜索相关主题资料（Tavily/Brave Search）
- 提取关键数据、趋势、案例
- 自动整理成引用格式

### Step 4：智能制作PPTX
- 调用 pptx-generator skill 生成可编辑PPTX
- 按选定风格排版（配色/字体/布局）
- 插入图表和资料
- 保存到指定目录

## 核心脚本

```
~/.openclaw/workspace/skills/ppt-smart-generator/scripts/
├── ppt_generator.py          # PPT生成核心引擎
└── chart_generator.py        # 图表数据生成模块
```

## 调用方式

**方式一：三先生直接说需求**
```
"帮我做一个工作汇报PPT，主题是Q1季度销售总结"
"生成一个项目立项提案，关于新办公楼装修"
"制作一个融资路演PPT"
```

**方式二：指定详细参数**
```
/ppt-generate --type 工作汇报 --scene 季度汇报 --topic "Q1销售总结" --search --output ~/output.pptx
```

## 图表类型速查

| 图表名 | 类型 | 适用场景 |
|--------|------|----------|
| 进度对比 | 柱状图 | 计划值 vs 实际值 |
| 完成率环形 | 环形图 | 任务完成率占比 |
| 趋势折线 | 折线图 | 时间序列变化 |
| 甘特图 | 横向柱状 | 项目实施时间线 |
| 增长曲线 | 折线图 | 融资/用户增长 |
| 受众画像 | 雷达图 | 用户特征分析 |

## 风格与配色

| 风格 | 主色 | 适用场景 |
|------|------|----------|
| 商务蓝 | #1E3A5F | 工作汇报、项目立项、经济分析 |
| 科技深 | #0D1B2A | 数据分析、技术分享 |
| 学术白 | #2C3E50 | 培训课件、学术报告 |
| 创意紫 | #6C3483 | 产品介绍、融资路演、策划方案 |

## 注意事项

- 生成前先询问三先生确认主题和风格偏好
- 页数控制在10页以内，每页聚焦一个核心观点
- 图表优先展示关键数据，不要堆砌
- 网络资料搜集用 Tavily，尊重时效性（24小时内）

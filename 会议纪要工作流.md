# 📋 会议纪要智能工作流

> 语音转文字 → 智能摘要 → 生成待办 → 创建飞书任务

---

## 工作流程

| 步骤 | 操作 | 工具 |
|------|------|------|
| 1️⃣ | 会议录音 | 手机/录音笔 |
| 2️⃣ | 语音转文字 | Whisper AI |
| 3️⃣ | 智能生成纪要 | Meeting Notes Pro |
| 4️⃣ | 提取待办事项 | AI自动识别 |
| 5️⃣ | 创建飞书任务 | 飞书API |

---

## 使用方法

### 方法一：上传录音文件
```
"帮我转录会议录音"
```
然后上传音频文件，我会：
1. 用Whisper转录为文字
2. 智能分析生成会议纪要
3. 提取待办事项
4. 帮您创建飞书任务

### 方法二：直接粘贴文字
```
"根据以下会议记录生成纪要：
[粘贴会议记录]
"
```

### 方法三：实时语音转文字
在Mac上可以直接说：
```
"用Whisper转录"
```

## 待办事项识别关键词

AI会自动识别以下句式的任务：

| 句式 | 示例 |
|------|------|
| "XXX负责..." | 张三负责整理会议纪要 |
| "请XXX..." | 请李四跟进客户反馈 |
| "XXX会..." | 王五会在周五前完成方案 |
| "需要XXX..." | 需要赵六协调资源 |
| "下次会议..." | 下次会议讨论预算问题 |

---

## 飞书任务创建

生成的待办事项可以一键创建为飞书任务：

```
待办任务：整理会议纪要
负责人：张三
截止日期：4月20日
```

→ 自动创建飞书任务
